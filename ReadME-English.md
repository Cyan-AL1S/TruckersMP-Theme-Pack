**Alky Design UI Installation Guide**  

**For the Old Version of the TruckersMP Launcher:**  
1. Download the “Alky Design UI” ZIP file, then right-click and extract its contents:  
   - `ats` – mod for American Truck Simulator (ATS) UI package  
   - `ets2` – mod for Euro Truck Simulator 2 (ETS2) UI package  
   - `shared` – base package (required)*  

2. Open the installation folder: `C:\ProgramData\TruckersMP\data`. How to find this folder?  
   2.1 Open “This PC”, click “View”, and check the “Hidden items” option.  
   2.2 Copy and paste the extracted files into the `C:\ProgramData\TruckersMP\data` folder.  

3. Open the TruckersMP plugin and click to launch the game. If you see a change in the interface, the installation was successful. If it doesn’t work, seek help from others.  

*End of the Old Version Plugin Section.*  

**For the New Version of the TruckersMP Launcher:**  
1. Download the “Alky Design UI” ZIP file and extract its contents:  
   - `ats` – mod for American Truck Simulator (ATS) UI package  
   - `ets2` – mod for Euro Truck Simulator 2 (ETS2) UI package  
   - `shared` – base package (required)*  

2. Open the new launcher and click “Settings”.  
3. Select the TruckersMP installation path and copy it.  
4. In the opened folder, copy and paste the extracted files.  
5. Open the folder named “data”; copy and paste the files into this folder as well.  
6. Open the TruckersMP plugin and click to launch the game. If the interface changes, the installation was successful. If it doesn’t work, seek help from others.  

*End of the New Version Plugin Section.*
