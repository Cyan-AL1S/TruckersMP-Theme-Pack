Alky Design UI 安装教程


旧版 TruckersMP 启动器：

1, 下载 ’Alky Design UI ’压缩包，鼠标右键单击解压
ats– mod=美卡 UI 包
ets2– mod=欧卡 UI 包
shared–mod=基础包 必装*

2，打开安装路径文件夹- C:\ProgramData\TruckersMP\data如何找到这个文件夹？
2.1 打开此电脑，单击“查看 ”，勾选“隐藏的项目 ”

2.2 将解压出的文件复制粘贴进路径 C:\ProgramData\TruckersMP\data

3，打开 TruckersMP 插件，点击启动游戏。如看到界面改变

即代表成功，如未生效寻求他人帮助

旧版插件篇结束

5，打开第一个文件夹（名称“data”）

6，将下载出来解压好的文件复制粘贴进此文件夹
